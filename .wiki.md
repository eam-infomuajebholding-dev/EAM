# Project Summary
This project is a web application designed to provide consultation services, enabling users to submit requests and messages that are stored in a database. It automates confirmation notifications for successful submissions and integrates with Atoms Cloud for effective data management. Recent enhancements include a refined user interface with a smooth light/dark mode transition, an AI chatbot for real-time assistance, and improved visual effects for user engagement, including elegant hover effects on service cards.

# Project Module Description
- **Backend**: Responsible for data storage, processing, and business logic.
- **Frontend**: User interface for submitting consultation requests and messages, featuring an AI chatbot and a seamless light/dark mode toggle.
- **Notifications Service**: Sends confirmation notifications to users.
- **Database**: Stores user submissions and related data.

# Technology Stack
- **Backend**: Python, FastAPI, SQLAlchemy
- **Frontend**: React, TypeScript, Tailwind CSS
- **Database**: PostgreSQL (via Alembic for migrations)
- **Notification Service**: Atoms Cloud

# Usage
1. Install backend dependencies:
   ```bash
   cd app/backend
   pip install -r requirements.txt
   ```
2. Install frontend dependencies:
   ```bash
   cd app/frontend
   pnpm install
   ```
3. Run database migrations (if applicable):
   ```bash
   alembic upgrade head
   ```
4. Start the backend server:
   ```bash
   cd app/backend
   uvicorn main:app --reload
   ```
5. Start the frontend development server:
   ```bash
   cd app/frontend
   pnpm run dev
   ```
